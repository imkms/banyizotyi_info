Azure Page

Ez az oldal a Azure témakört mutatja be. Itt találhatók információk a Azure rendszerről, parancsokról, disztribúciókról, és egyéb hasznos tudnivalókról.

Feladat: WordPress App Service létrehozása

1. Lépj be az Azure Portálra. 2. Kattints az „Erőforrás létrehozása” gombra. 3. Keresd meg: WordPress → válaszd a WordPress on App Service sablont. 4. Erőforráscsoport neve: VIZSGA 5. Régió: Nyugat-Európa 6. Weboldal neve: wp-monogram.azurewebsites.net 7. App Service-csomag: F1 (Free) 8. MySQL adatbázis: Burstable, ingyenes 9. Webhely nyelve: magyar 10. Admin felhasználónév: monogram 11. Admin jelszó: P@ssw0rd

Feladat: WordPress felhasználó létrehozása

1. Jelentkezz be a WordPress admin felületére. 2. Felhasználók → Új hozzáadása. 3. Név: monogram (ékezetek nélkül), jelszó: P@ssword 4. Szerepkör: Szerkesztő 5. Hozzáadás mentése

Feladat: Honlap alapbeállítások módosítása

1. Beállítások → Általános. 2. Honlap neve: Monogram 3. Rövid leírás: a nagy világjáró 4. Mentés

Feladat: WordPress sablon testreszabása

1. Megjelenés → Sablonok → Aktiváld: Twenty Thirteen 2. Megjelenés → Testreszabás 3. Fejléc szöveg színe: #1E324A 4. Fejléc képe: PanoramaBerlin2021.jpg (középen legyen az út)

Feladat: Oldal létrehozása Berlin címmel

1. WordPress admin felületen: Oldalak → Új hozzáadása 2. Cím: Berlin 3. Kiemelt kép: Berlin_Montage_4.jpg 4. Szöveg: A Wikipedia „Berlin” szócikkének első 2 bekezdése

Feladat: Virtuális gép létrehozása Azure-ban

1. Lépj az Azure Portálra → Erőforrás létrehozása → Virtuális gép. 2. Erőforráscsoport: VIZSGA 3. Gépnév: WS2019-monogram 4. Régió: Nyugat-Európa 5. Kép: Windows Server 2019 Datacenter – x64 Gen2 6. Méret: 2 vCPU, 8 GB RAM 7. Felhasználónév: monogram 8. Jelszó: P@ssw0rdP@ssw0rd 9. Engedélyezett port: Távoli asztal (RDP)

Feladat: Virtuális hálózat létrehozása

1. Virtuális hálózat neve: bármilyen (pl. vh-monogram) 2. Hálózati cím: 192.168.17.0/24 3. Alhálózat neve: monogram 4. Alhálózati cím: 192.168.17.32/28

Feladat: IP és adapter törlése VM törlésével

1. VM létrehozásakor pipáld be: „A nyilvános IP-cím és hálózati adapter törlése a virtuális gép törlésekor”.

Feladat: Automatikus leállítás beállítása

1. VM → Műveletek → Automatikus leállítás 2. Időpont: 11:00 (CET) 3. Értesítés: Nem szükséges 4. Mentés

Feladat: Távoli asztali kapcsolat tesztelése

1. Indítsd el a VM-et 2. Kattints: Csatlakozás → RDP → RDP-fájl letöltése 3. Nyisd meg, jelentkezz be a felhasználónév/jelszó párossal 4. Server Manager → Helyi kiszolgáló megnyitása

Feladat: Google Drive mappa létrehozása és megosztása

1. Nyisd meg a Google Drive-ot 2. Új → Mappa → Név: monogram 3. Jobb gomb → Megosztás → Add hozzá saját ms.gdszeged.hu fiókodat szerkesztőként 4. Mentés

Feladat: Dokumentum létrehozása a mappában

1. A monogram mappán belül kattints: Új → Google Dokumentum 2. Cím: monogram 3. Tartalom: Megennék egy kakaós csigát, meginnék hozzá egy pohár kakaót.

Feladat: Azure erőforrások leállítása és törlése

1. WordPress App Service → Leállítás 2. Töröld: MySQL adatbázis, App Service, App Service-csomag, Felügyelt identitás 3. Virtuális gép → Leállítás → Törlés 4. Töröld a hálózatot és a hálózati biztonsági csoportot 5. Végül töröld az egész VIZSGA nevű erőforráscsoportot 6. Jelentkezz ki az Azure-ból
